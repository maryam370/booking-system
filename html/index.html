About OpenSphere

OpenSphere is a free and open-source platform designed to facilitate collaborative software development. Originally started as an ITSE305 class project, it has grown into a community-driven initiative.

Project Name Origin

The name "OpenSphere" was chosen to represent our vision of creating an open, inclusive environment (sphere) where developers and users can collaborate freely.

Project Status

Current development status: Alpha - Core features are implemented but still undergoing testing.
